# Screenshots and Images

## How to Add Screenshots

1. **Run the application**:
   ```powershell
   python gui/app_modern.py
   ```

2. **Take screenshots** of:
   - Main application window
   - Browse dialog (with tabs)
   - Copy operation in progress
   - Statistics window
   - Network device selection

3. **Save screenshots** in this folder with these names:
   - `main-interface.png` - Main application window
   - `device-selection.png` - Browse dialog with Local/Network/USB tabs
   - `copy-progress.png` - Copy operation with animated dashboard
   - `statistics.png` - Statistics window
   - `network-support.png` - Network device UNC path entry

4. **Recommended tools** for screenshots:
   - Windows: Win + Shift + S (Snipping Tool)
   - Or use: Snagit, ShareX, Greenshot

## Screenshot Guidelines

- **Resolution**: 1920x1080 or higher
- **Format**: PNG (preferred) or JPG
- **Content**: Show actual UI, not mockups
- **Annotations**: Add arrows/highlights if needed
- **File size**: Keep under 500KB each (optimize if needed)
